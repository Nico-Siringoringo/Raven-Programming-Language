# Bahasa Programming Language

## Installation

1. Download the appropriate executable from the release assets.
2. Place the executable in your project directory.

## Usage

Run a `.john` file with:

### Linux
```bash
./john <your-file>.john
```

Example:

```bash
./john main.john
```

### Windows
```bash
john.exe <your-file>.john
```

Example:

```bash
john.exe main.john
```
